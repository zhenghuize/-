## 四根小黄瓜的处女作
> 这个项目是一个“游戏社区” 它的由来，就是闲，无聊
+ 要想启动此项目，就克隆地址，然后👇
> git clone https://github.com/zhenghuize/taptap.git
+ 打开 admin 文件  启动终端：cmd （不要关闭服务）
> npm install | yarn install
> node server
+ 然后返回上一个文件 启动终端
> npm install | yarn install
> yarn serve
+ 如果你要打包
> yarn build
